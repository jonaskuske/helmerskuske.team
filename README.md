### Instructions

Install dependencies:  
`> npm install`

Start dev server on [`localhost:1234/index.html`](http://localhost:1234/index.html):  
`> npm start`

Build for production:  
`> npm run build`

Deploy to GitHub Pages:  
`> npm run deploy`
